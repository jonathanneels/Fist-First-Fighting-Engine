<h1>Fist First Fighting Engine</h1>
<p><strong>The 21st-Century Cross-Platform M.U.G.E.N Alternative</strong>
</p>
<p><strong>A fully working demo is available here (including spritesheets and an example .fistf file for import into the engine): </strong><a href="https://jonathanneels.itch.io/combolade"><strong></strong></a><strong><a href="https://jonathanneels.itch.io/combolade">https://jonathanneels.itch.io/combolade</a></strong><em></em><em></em></p>
<img width="382" height="187" alt="tiny_movie" src="https://github.com/user-attachments/assets/5b023267-48b2-456c-949d-b40b9424a9d7" /><br>
<p>Welcome to the <em>Fist First Fighting Engine</em>. Are you dreaming of creating your own fighting game but don't want to deal with bloated software, complex installations, or legacy platforms? Look no further.
</p>
<p><em>Fist First</em> is a lightning-fast, web-native 2D fighting engine designed for accessibility, speed, and pure creative freedom. Whether you are playing on a PC with a gamepad or on a smartphone via touch controls, your game runs everywhere. Directly in the browser.
</p>
<h3>Why Choose Fist First?</h3>
<ul><li><strong>Simple, Fast & Efficient:</strong> Forget about long, tedious tutorials. Create your own fighters, stages, movesets, and hitboxes via the built-in visual editors. Export your data as JSON or publish your game as a standalone HTML file with a single click. You can use classic spritesheets if you want to create a more refined title.&nbsp;</li><li><strong>Custom Plugins:</strong> Your game, your rules. The engine supports <em>custom JavaScript plugins</em>. Easily inject your own code and logic to tailor the engine to your vision without breaking the core architecture.
</li><li><strong>Cross-Platform by Design:</strong> Built to run on desktop, tablet, and mobile. Full support for keyboard, touchscreens (including an on-screen D-pad), and seamless Gamepad API integration.
</li><li><strong>Modern Features:</strong> Packed with modes like Arcade, Time Attack, Survival, PvAI, PvP, and AI vs. AI. Includes advanced systems for dynamic weather, "travelcubes," projectiles (and in a future release stable network multiplayer).
</li><li><strong>Made by a fighting game fanatic: recoveries, super jumps, cancels, dashes, and counter-grabs</strong>. All the things that make fighting games special are packed into this project</li><li><strong>Cool things</strong>: set <strong>weather and camera filters</strong> for each stages <strong>and set a movie as background</strong>,&nbsp;for each character you can <strong>set the victory points condition for unlock</strong> (no dlc, just playing the game), set characters as boss in the arcade/time attack modes, <strong>make a smash bros brawler, a close-up fighter, a ring out fighter or a multi-stage fighter</strong> (thanks to the teleporter cube). It is <strong>very mobile friendly</strong>, so low-end devices can join the fun.&nbsp;<strong>There is a lane-switch mode</strong> (like Fatal Fury&nbsp;games).&nbsp;&nbsp;So much freedom!</li></ul>
<h3>Open-Source & "The Creator's Clause" (License)</h3>
<p><em>Fist First Fighting Engine</em> is open-source and built to inspire the community. You are free to use, modify, and build your dream game. However, there are two unique and strict requirements for <strong>commercial use</strong>:
</p>
<ol><li><strong>Attribution is Mandatory:</strong> If you monetize a game running on this engine, the original creator (Jonathan Neels) must be clearly and respectfully credited and pay the modest "commercial royalty".
</li><li><strong>The "VIP Invite" Clause:</strong> Should your game skyrocket in popularity and become a massive commercial success, that’s fantastic! However, I expect a personal invitation to your official launch party, tournament, or press event<strong>&mdash;</strong><strong>including paid travel and accommodation</strong>. Deal?</li></ol>
<p><em>Ready to build the next generation of fighting games? Fork the repo, open the engine, and throw that first punch!</em>
</p>
